== Description ==
A comprehensive booking management plugin for WordPress.

== Installation ==
1. Upload the `booking-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Booking Pro > Settings to configure the plugin.

== Frequently Asked Questions ==

= How do I add new services? =
You can add new services under Booking Pro > Services in the WordPress admin panel.

= Does Booking Pro support multiple languages? =
Yes, Booking Pro can be translated into multiple languages.
